# SAS_Scraper
Tool for scraping NHL player data

# Usage
1. Input file: .txt file with player names one per line.
Example file  
Joe Pavelski  
Claude Giroux  
Alex Ovechkin  
...  
  
2. Install dependencies  
python setup.py Install  

3. Run command line  
Example Command Line  
python nhl.py -i players.txt
